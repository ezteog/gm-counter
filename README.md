# GM Counter ☀️

A minimal, onchain-themed **GM counter** — a tiny web app to track your daily "GM" (good morning), styled after the [Base](https://base.org) visual identity.

**Live demo:** https://ezteog.github.io/gm-counter/

---

## Features

- ☀️ One-tap **GM** button with a running total
- 💾 **Persistent count** — saved in the browser (localStorage), so it never resets on refresh
- 🔥 **Daily streak** counter (consecutive days you say GM)
- 🕒 Shows the **date & time of your last GM**
- ♻️ **Reset** button with a confirmation prompt
- 🟦 **Base-inspired** theme (Base Blue, clean geometric look)

## How it works

100% front-end: plain **HTML, CSS and vanilla JavaScript** — no framework, no backend. Your data is stored locally in the browser with `localStorage`, so the counter persists between visits on the same device.

## Project structure

- `index.html` — page structure
- `style.css` — styles (Base-inspired theme)
- `script.js` — counter, streak and persistence logic
- `favicon.svg` — Base-style icon

## Run locally

Clone or download the repository and open `index.html` in your browser. That's it — no build step, no dependencies.
